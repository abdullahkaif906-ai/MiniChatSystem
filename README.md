# Mini Chat System

A console-based chat application developed in Java.

## Features

- Send messages
- Automatic bot replies
- View chat history
- Search messages
- Delete messages
- Count total messages
- Clear chat history
- Save chat to a text file
- Time-stamped messages

## Technologies Used

- Java
- FileReader
- FileWriter
- Arrays
- Methods
- Object-Oriented Programming

## How to Run

1. Compile the Java file.
2. Run the program.
3. Use the menu to interact with the chat system.

## Future Improvements

- User login system
- Multiple chat rooms
- GUI using Java Swing
- Better AI responses
- Chat encryption
